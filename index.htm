Save for Good

Save for Good is a single page application developed to allow the funding of loans to entrepreneurs around the world, through the non-profit Maple Microdevelopment. Built on the most modern web development patterns it incorporates React and Redux for it's front end and Ruby on Rails to support the back end.

Samwise Storefront Management

Samwise is a storefront management system. Currently in production it has already processed nearly 2,000 orders collectivily worth over $400,000. Built on Laravel and Vue.js, it manages all aspects of a modern buisness such as billing (Stripe API), Shipping (EasyPost API), and realtime searching (ElasticSearch Engine).

Pangolin 4x4

This is a single page storefront web app built for Series Land Rover restoration shop Pangolin 4x4. Built on the Samwise Storefront Management system and Vue.js. It incorperates modern web development techiques and tools to allow for effeciantly finding parts from the thousands of parts in Pangolin 4x4's inventory.

A-Maze

A study in maze algorithms, "A-Maze" is both a maze generation and solution algorithm visualizer. Incorporating a hand-rolled graph to canvas rendering engine, React/Redux controls, classic maze algorithms, and functional language paradigms. It lets anyone see the many different methods to programmatically develop or solve mazes.
